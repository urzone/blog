Cloud Flare 重定向

XiaO / 2023-05-09

网站域名发生变更后，搜索引擎所收录的旧地址，通通变成了 404. 这很影响网站访问者的心情啊。

域名托管

新域名托管到 Cloud Flare 后，即可通过配置规则对旧地址重定向到新的域名。

重定向

具体操作：选中托管的域名 –> 左边栏 Rules –> Page Rules –> Create a Page Rule

URL （required）

Forwarding URL, 301 - Permanent Redirect

Enter destination URL (required)

如果新旧网站的内容结构未发生变化的话，可使用通配符 * 重定向所有的旧地址，在新地址中使用 $1 来匹配第一个字符。

HTTP 状态检测

使用该工具可检测网页的跳转情况。

Google Search Console

在 Google Search Console 中添加站点的 sitemap，使机器人重新爬取网站信息，以此增加检索。

Google Tag Manager

有人说，在页面添加相应的代码，会影响特殊地区的加载。有人说异步加载，并不影响加载速度。想了想，还是直接砍了吧。反正除了域名外，一切都是免费的，那就索性都免费吧。
